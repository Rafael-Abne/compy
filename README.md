### Analisa preços de produtos de forma automatica. :computer:
Ao digitar o produto que deseja, esse script busca o mesmo por preços mais baratos.

<hr>
 
Uso
 ------------
 
 - Clone o projeto.
 ```sh
 git clone https://github.com/fabriciodearaujo/comparar-precos
 ```
 - Rode o script.
  ```sh
 python app.py
 ```
